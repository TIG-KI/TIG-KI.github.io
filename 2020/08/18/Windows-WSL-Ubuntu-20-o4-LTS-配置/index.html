其他

  1. Ubuntu 20.04 中无 pip 可通过下面的命令进行快捷安装
  2. 如果用 WLS 自带的窗口对 WSL 进行操作会产生令人讨厌的错误提示音，但是使用 Windows terminal 对WSL 进行操作时就不会出现该问题。
  3. 在使用 Windows terminal 操作 WSL 时默认打开路径为 /mnt/c/Users/{ 当前 PC 的用户} , 要想设置 Windows terminal 打开 WSL 直接进入 Ubuntu 的用户目录的解决办法可参考 通过Windows Terminal打开Linux子系统直接进入家目录
